# Welcome to File Explorer

This is a sample markdown file demonstrating the code editor plugin.

## Features

- Windows 11 style UI
- Plugin-based architecture
- Support for multiple file types
- Authentication system

## Supported Plugins

1. **Auth Plugin** - User authentication and authorization
2. **Image Viewer** - View images in various formats
3. **Code Editor** - Edit code files with syntax highlighting
4. **PDF Viewer** - View PDF documents
5. **Office Viewer** - View Word, Excel, and PowerPoint files

## Usage

Simply navigate through folders and click on files to view/edit them.
